<h1>Video widget in Wordpress</h1>

在 wordpress 的 widget 中塞入 member 資料，方便管理者輕鬆管理<br>
盡量減少使用者寫code，讓一般人都可以輕鬆上手<br>
努力加強更多社交網站的 icon ，會以SVG的方式呈現，增加美觀

目前支援
> 成員名稱，成員信箱<br>
> 成員 facebook google+ twitter url
